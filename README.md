# 夏やすみボード

家族3人で使う夏休み用チェックアプリです。ポイント機能はありません。

オンライン版: https://cotoiro.github.io/summer-app/

## 確認方法

このフォルダでローカルサーバーを起動し、表示されたアドレスをブラウザで開きます。

```sh
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開いてください。

## 入っている機能

- 子ども2人の切り替え
- 日付ごとの宿題・勉強、お手伝いチェック
- 曜日を指定した「やること」の追加・編集・削除
- 家族の予定・メモの追加・削除
- 月間カレンダー
- ICS形式でのカレンダー読み込み・書き出し
- JSON形式での全データのバックアップ・復元
- ブラウザ内への自動保存
- PC、タブレット、スマホ幅に合わせた表示

## 保存と共有

- 家族共通のメールアドレスとパスワードでログインします。
- 予定・やること・チェック記録はSupabaseへ保存し、PC・スマホ・タブレットで共有します。
- 通信できない場合に備えて、現在の端末にもデータを保存します。
- 通知、ポイント、ごほうび機能はありません。

## Googleカレンダーとの予定の受け渡し

- アプリからGoogleカレンダーへ：カレンダー画面の「ICSを書き出す」でファイルを保存し、Googleカレンダーの「設定」→「インポート／エクスポート」から読み込みます。
- Googleカレンダーからアプリへ：Googleカレンダーから予定を書き出し、カレンダー画面の「ICSを読み込む」で `.ics` ファイルを選びます。
- Googleカレンダーから複数カレンダーをまとめて書き出した場合はZIPファイルになるため、先に開いて中の `.ics` ファイルを選んでください。
- 同期ではないため、読み込み後の変更は自動では反映されません。同じICSを再度読み込んだ場合、登録済みの予定はできるだけ重複しないように判定します。


オンライン化の準備内容は `ONLINE_SETUP.md`、データベース作成SQLは `supabase/schema.sql` にあります。
